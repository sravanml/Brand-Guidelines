# Sri Sathya Sai Center of Austin — Design System

Official design system for the Sri Sathya Sai Center of Austin.

## Colors
- **Royal Blue** `#1A5EA8` — Primary brand color
- **Saffron Orange** `#F7941D` — Accent color
- **Warm Neutral** `#F9F7F4` — Backgrounds & text

## Fonts
- Cinzel — Display headings
- Source Serif 4 — Editorial body text
- Jost — UI elements

## Deploy
Hosted on Vercel. Open `index.html` to view locally.
